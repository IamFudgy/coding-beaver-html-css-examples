ถึง Coding Beaver ในอีก 3 เดือนข้างหน้า

เรารู้ว่านายคงเหนื่อย และอยากจะยอมแพ้ในการเปลี่ยนแปลงตัวเองจนแทบจะทนไม่ไหว แต่เราอยากจะบอกนายว่า อย่ายอมแพ้เด็ดขาด

ลองมองย้อนกลับมาดูตอนที่ริเริ่มความตั้งใจสิ แล้วดูว่าตอนนี้ตัวเองมาได้ไกลแค่ไหนแล้ว อย่ายอมแพ้ ความฝันอยู่อีกแค่เอื้อมมือเท่านั้น

ไม่มีอะไรที่ได้มาโดยง่าย ทุกอย่างล้วนต้องผ่านความอดทนและความพยายามทั้งนั้น คนที่มีประสบความสำเร็จในวันนี้ล้วนเป็นคนที่เคยไม่สำเร็จมาก่อนทั้งนั้น
