การพัฒนา Front-end

ทำไม Web developer ถึงควรเรียน Front-end ก่อน?

การพัฒนา Front-end ถือเป็นพื้นฐานที่สำคัญสำหรับการเรียนรู้ในเส้นทางของ Web developer เพราะว่า จะทำให้ทุกคนได้รับการปูพื้นฐานจากสิ่งที่จับต้องได้ก่อน แล้วค่อยๆเรียนรู้การทำงานร่วมกันของเว็บ ส่วนหน้าบ้านกับส่วนหลังบ้าน ที่เชื่อมโยงถึงกันผ่าน Internet

ภาษา Front-end ที่ต้องเรียน

HTML: Hypertext Markup Language

HTML เป็นส่วนประกอบที่เป็นรากฐานของ Front-end เพราะว่า HTML คือโตรงสร้างของเว็บ เป็นตัวกำหนดว่าอะไรควรอยู่ตรงไหน เรียงลำดับยังไงบ้าง

CSS: Cascading Style Sheet

CSS คือภาษาที่ใช้สำหรับตกแต่ง HTML ให้สวยงามมากยิ่งขึ้น เพื่อเพิ่มประสบการณ์การใช้งานหรือ User's Experience ให้ดียิ่งชึ้นเวลาเข้าชม Website ของเรา

JavaScript

JavaScript เป็น Programming Language ที่ Web browser เข้าใจ สามารถทำสิ่งต่างๆที่ซับซ้อนภายใน Web ของเราได้ สามารถติดต่อสื่อสาร และทำงานร่วมกันกับระบบหลังบ้านหรือ Back-end ผ่านทาง Internet
